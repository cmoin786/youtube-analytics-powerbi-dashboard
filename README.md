# YouTube Analytics Dashboard

## Project Overview

This project presents an interactive Power BI dashboard built to analyze YouTube trending video performance across different categories, countries, and channels.

The dashboard helps identify content trends, audience engagement patterns, and category performance using visual analytics.

---

## Objectives

* Analyze total video views.
* Track audience engagement.
* Identify top-performing categories.
* Compare country-wise video performance.
* Monitor channel performance.
* Explore trending patterns over time.

---

## Tools Used

* Power BI
* DAX
* Power Query
* Data Modeling
* Data Visualization

---

## Dashboard Pages

### 1. Overview

KPIs:

* Total Views: 31.17 Billion
* Total Videos: 10K
* Average Engagement: 7.58

Visuals:

* Total Views by Year
* Total Views by Category
* Country Filter
* Publish Date Filter

### 2. Category Analysis

* Category-wise views
* Category-wise engagement
* Top-performing content categories

### 3. Country Analysis

* Country-wise trending performance
* Regional content popularity

### 4. Channel Analysis

* Top channels by views
* Subscriber and engagement analysis

### 5. Trend Analysis

* Yearly growth trends
* Trending content patterns

---

## Key Insights

* Entertainment generated the highest number of views.
* Gaming and Sports ranked among the most viewed categories.
* Viewer engagement remained strong across major categories.
* Trending video performance varied significantly across countries.
